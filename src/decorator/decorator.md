# Decorator Pattern
동적으로 책임 추가가 필요할 때 사용하는 패턴

### 커피 제조 방법
- 에스프레소 : 커피의 기본
- 아메리카노 : 에스프레소 + 물
- 카페라떼 : 에스프레소 + 스팀밀크
- 헤이즐넛 : 아메리카노 + 헤이즐넛 시럽
- 카페모카 : 카페라떼 + 초콜릿
- 카라멜 마끼아또 : 카페라떼 + 카라멜 시럽

### 구현
실제 기능의 베이스가 될 인터페이스를 만들고, 이를 구현한다.\
데코레이터도 같은 인터페이스를 상속 받아서 구현하되, 다른 구현체를 생성자의 매개변수로 받아서 사용한다.\
데코레이터는 입력 받은 다른 구현체에 추가적인 기능을 더해서 같은 인터페이스 타입을 반환하게 된다.\
이로써 기존 구현체에 다른 기능을 더한 구현체를 계속해서 만들 수 있는 구조가 완성된다.
